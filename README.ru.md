# File DB Manager

Профессиональный инструмент для индексации и управления файлами с SQLite.  
Сканируйте папки, отслеживайте файлы, проверяйте списки путей и разрешайте неоднозначности вручную.

---

## ✨ Возможности

- 📁 **Полнотекстовый индекс файлов** в SQLite (имя, путь, размер, хэш, дата изменения)
- 🔍 **Нечёткий поиск** с нормализацией Unicode (работает с кириллицей и спецсимволами)
- 📊 **Управление наборами** — сравнение списков путей (буфер/TXT/Excel) с базой данных
- 🎯 **Три статуса**: `FOUND` / `MISSING` / `AMBIGUOUS`
- 🌐 **Двуязычный интерфейс** (русский/английский) с переключением на лету
- ⚡ **Инкрементальные обновления** — не нужно полное пересканирование
- 🖥️ **Интеграция с Windows** — "Показать в папке" с выделением файла (Win10/Win11)
- 🧪 **Набор тестов** pytest (8 тестов, покрывающих основную логику)

## 🚀 Быстрый старт

### Требования
- Python 3.10 или выше

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Mishin-18/file-db-manager.git
cd file-db-manager

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите программу
python file_db_manager.py
